Node Monitor
Remote Monitoring & Control

Node Monitor is an open source component you add to your Node.js app, and a dashboard for visualizing and controlling your app servers at runtime.

To add monitoring, install node monitor onto your app servers: 

$ npm install -g monitor

Then add it to your application startup, and restart your app: 

var Monitor = require('monitor').start();

To monitor your app servers, install the dashboard (onto localhost for now): 

$ npm install -g monitor-dashboard
$ npm start monitor-dashboard

Now navigate to http://localhost:4200/ in your browser. The Getting Started page has more information about building dashboards using the built-in components.
